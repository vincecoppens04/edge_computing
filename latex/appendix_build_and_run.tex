This appendix summarises how to build the container image, import it into the local k3d cluster, and deploy the subscriber.

\section{Build the image locally}
\begin{lstlisting}[language=bash,caption={Build the subscriber image}]
cd edge_computing
docker build -t edgecloud-subscriber:latest .
\end{lstlisting}

\section{Run in a local k3d cluster}
Import the locally built image into the k3d cluster:
\begin{lstlisting}[language=bash,caption={Import image into a k3d cluster}]
k3d image import edgecloud-subscriber:latest -c edgecloud
\end{lstlisting}

Deploy the subscriber:
\begin{lstlisting}[language=bash,caption={Deploy the MQTT subscriber}]
kubectl apply -f edge_computing/subscriber-deployment.yaml
kubectl get pods -l app=mqtt-subscriber
kubectl logs -f deploy/mqtt-subscriber
\end{lstlisting}
